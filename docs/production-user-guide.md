# Production User Guide
## Azee - Donut Shop Management System

### 1. Overview

Production users are responsible for managing the entire production planning and manufacturing process in the Azee system. Your role is crucial for ensuring stores receive the right quantities of products at the right time while optimizing costs and minimizing waste.

**Key Responsibilities**:
- Create and manage daily production plans
- Manage donut varieties and forms
- Configure box packaging options
- Monitor production costs and profitability
- Track and analyze waste across all stores
- Optimize production efficiency

### 2. Access & Permissions

#### 2.1 Production Access
- **Production Planning**: Create and modify production plans
- **Product Management**: Manage varieties, forms, and box configurations
- **Cost Analysis**: View and analyze production costs
- **Waste Tracking**: Monitor waste across all stores
- **Store Production**: View all store production data
- **Analytics**: Access production-related statistics

#### 2.2 Restricted Access
- **User Management**: Cannot create or modify users
- **Store Configuration**: Cannot modify store settings
- **System Settings**: Cannot change global configurations
- **Admin Functions**: No access to admin-only features

### 4. Production Planning

#### 4.1 Creating Production Plans
**Navigation**: Dashboard → Production → Create Plan

**Planning Process**:
1. **Select Date**: Choose production date
2. **Store Selection**: Select stores for production
3. **Product Planning**:
   - Choose donut varieties
   - Select donut forms (ring, filled, etc.)
   - Specify quantities per variety per store
   - Configure box packaging

**Planning Considerations**:
- Historical demand data
- Seasonal variations
- Store-specific preferences
- Weekend vs weekday patterns
- Special events or promotions

#### 4.2 Multi-Store Planning
**Bulk Planning Features**:
- **Template Creation**: Save common planning templates
- **Bulk Apply**: Apply same plan to multiple stores
- **Proportional Scaling**: Scale quantities based on store size
- **Regional Variations**: Adjust for local preferences

**Store-Specific Adjustments**:
- Individual store requirements
- Special dietary needs
- Local demand patterns
- Delivery constraints

#### 4.3 Production Plan Management
**Available Actions**:
- **Edit Plans**: Modify existing plans before production
- **Duplicate Plans**: Copy successful plans to new dates
- **Delete Plans**: Remove cancelled production plans
- **Export Plans**: Generate production sheets for staff

### 8. Production Workflows

#### 8.1 Daily Production Workflow
**Morning Routine** (6:00 AM - 8:00 AM):
1. Review today's production plans
2. Check ingredient inventory levels
3. Verify staff scheduling
4. Confirm delivery schedules
5. Address any overnight issues

**Production Planning** (8:00 AM - 10:00 AM):
1. Create or adjust tomorrow's production plans
2. Review store requests and feedback
3. Analyze yesterday's performance
4. Update cost calculations

**Monitoring** (Throughout Day):
- Track production progress
- Monitor quality control
- Handle urgent store requests
- Update delivery confirmations

**End of Day** (5:00 PM - 6:00 PM):
1. Review daily production results
2. Update waste reports
3. Prepare next day's schedules
4. Generate daily summary reports

#### 8.2 Weekly Production Workflow
**Monday - Planning Week**:

**Wednesday - Mid-Week Review**:
